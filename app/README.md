# StickerMaker

表情包自动制作工具

## 技术栈

- GUI界面:https://qfluentwidgets.com/
- 打包工具:pyinstaller
- 背景去除:https://github.com/danielgatis/rembg
- 水印去除:https://github.com/advimman/lama
- 视频关键帧提取:ffmpeg.exe
- 图片处理:https://pypi.org/project/pillow

## 目标


#### 模式一: 成套静态表情包

1. 拖入单张、或多张3*3、3*4、4*3的九宫格、十二宫格图片，或拖入参考图片，通过大模型接口生成单张、或多张3*3、3*4、4*3的九宫格、十二宫格图片（生成先留空）（3*3、3*4、4*3手动选择
2. 去除图片右下角文字水印(可选)
3. 去除背景，背景设置成透明
4. 根据3*3、3*4、4*3自动切割图片，逐行逐个编号分割后的文件
5. 对每个分割后的文件进行处理保存
    - 从左从右横向/从上从下纵向扫描，碰到非透明像素后退5个像素停止，提取中间图像，最后生成max(height,width)*max(height,width)的正方形图片，宽或者高不足的地方向两边透明填充
    - 伸缩为240*240透明png保存

#### 模式二: 单个动态表情包

1. 拖入单张、或多张3*3、3*4、4*3的九宫格、十二宫格图片，或拖入参考图片，通过大模型接口生成单张、或多张3*3、3*4、4*3的九宫格、十二宫格图片（生成先留空）（3*3、3*4、4*3手动选择
2. 去除图片右下角文字水印(可选)
3. 去除背景，背景设置成透明
4. 根据3*3、3*4、4*3自动切割图片，逐行逐个编号分割后的文件
5. 对每个分割后的文件进行处理保存
    - 从左从右横向/从上从下纵向扫描，碰到非透明像素后退5个像素停止，提取中间图像，最后生成max(height,width)*max(height,width)的正方形图片，宽或者高不足的地方向两边透明填充
    - 伸缩为240*240透明png保存
6. 按照编号将png图片合并成无线循环gif保存

#### 模式三: 通过视频成套动态表情包

1. 拖入单个3*3、3*4、4*3的九宫格、十二宫格布局的视频（3*3、3*4、4*3手动选择）
2. 从视频抽取png关键帧，进行图片序列编号。
3. 去除图片右下角文字水印(可选)
4. 去除背景，背景设置成透明
5. 根据3*3、3*4、4*3自动切割图片，逐行逐个按照 图片序列编号_行_列 编号分割后的文件
5. 对每个分割后的文件进行处理保存
    - 从左从右横向/从上从下纵向扫描，碰到非透明像素后退5个像素停止，提取中间图像，最后生成max(height,width)*max(height,width)的正方形图片，宽或者高不足的地方向两边透明填充
    - 伸缩为240*240透明png保存
6. 将同图片序列编号的png图片序列合并成无线循环gif保存
